# Incident Runbooks

## Scope
This directory contains first-response and recovery procedures for high-impact operational incidents in SAT Graph RAG.

## Runbook Index
- `neo4j-unavailable.md`
- `oidc-jwks-outage.md`
- `model-endpoint-degradation.md`
- `migration-failure.md`

## Operating Rule
Each incident response must produce an auditable record containing:
- incident start and end timestamps,
- actions executed,
- system state before and after mitigation,
- follow-up action items with owners and deadlines.
